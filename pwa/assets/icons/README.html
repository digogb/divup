Opção 2: Manual

Redimensione o ícone 512x512 para os seguintes tamanhos:

  • icon-72.png (72x72)
  • icon-96.png (96x96)
  • icon-128.png (128x128)
  • icon-144.png (144x144)
  • icon-152.png (152x152)
  • icon-192.png (192x192)
  • icon-384.png (384x384)
  • icon-512.png (512x512) ✓ Já existe

Opção 3: ImageMagick (se instalado)

magick icon-512.png -resize 192x192 icon-192.png
magick icon-512.png -resize 144x144 icon-144.png
